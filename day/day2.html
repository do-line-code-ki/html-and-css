What is an HTML element ?

Every line of code that you write in html document stating with a < opening tag> and ending with a < closing tag > is called an element.

For example:

This line that you are reading is currently wrapped inside p tags.

So this whole line of code < p > This line that you are reading is currently wrapped inside p tags. < /p > is called an HTML element

This whole chunk of text is inside < body > element and < body > elements is inside < HTML > ELEMENT.

So what is tag ?

A tag is a part of an elements with 2 variants: opening tag and closing tag

So, do all tag have opening and closing tags ?

No, there are some tags known as Standalone tags like < br > (To add a new line). So it is best advised to not use tags without closing tags. Best practice is to close tags whenever you open one

Let's talk about attributes

Attributes are special qualities you add to an element. These qualities are declared after opening a html tag. For example:

this line that you are reading has a clickable link, this link is added by using href attribute inside an < a > tag.

Similarly, the lang="en" we use after opening the < html > tag is also an attribute.

Go back to the clickable line, did you notice something ? What?!!!!

We used an < a > tag inside a < p > tag, a tag inside a tag, this is called NESTING or NESTED TAGS.

When using nested tags it is necessary you close the inner tags first before closing the outer tag.

this is inner an outer tags have special names for them: parent tag (for outer tag) and child tag (for inner tag)

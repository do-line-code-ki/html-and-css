Day 4 - Body Element of HTML Document

Section

This is parent section of the document. Sections are used to divide the content of document into topic-based sections. If you have multiple topics on your page then you should section but if the topics are related to each other somehow use nested sections.

Footer of Article

How to decide when to use section and article element ?

Think it this way, section breaks down the whole document into topic specific sections, if these topics are mutually exclusive then you need to have multiple sections. If some topics are sub-topics of some topics then you need to have them nested under a single section. If these topics aka sections have pieces of content that have pieces of content that repeat in same pattern, use article.
